This is a sample Project for using own MsBuild Tasks

This sample include the package MazaBuildTasks that defines a Task "MazaTask" 
see the project UsingMazaBuildTask.csprojfile how to call it.
The file UsingMazaBuildTask\NuGet.config defines the local feed where the Project MazaBuildTasks.csproj store the nuget package,
you can select it in the PackageManager.


After changing the package MazaBuildTasks you must reload Visual Studio the get the new Version!
